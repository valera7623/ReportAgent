Частые вопросы

Какие файлы поддерживаются?

CSV, Excel (.xlsx), а также публичные Google Sheets по ссылке. Пример тестового файла доступен в API-документации.

Нужен ли OpenAI ключ?

Для AI-анализа и голосового ввода нужен ваш OPENAI_API_KEY при self-hosted развёртывании. На демо-сайте ключ уже настроен.

Есть ли API?

Да. Swagger: /docs. Создайте API-ключ в личном кабинете и вызывайте POST /generate_report.

Можно ли развернуть у себя?

Да. Docker Compose, GitHub Actions, документация на /help/. Подходит для своего SaaS или внутреннего инструмента.
