# Glossary

| Term | Description |
|------|-------------|
| **API key** | `ra_...` token for `X-API-Key` authentication |
| **Agent** | Pipeline module: parser, analyst, visualizer, etc. |
| **Celery** | Background task queue |
| **Intent** | Parsed voice request meaning |
| **JWT** | Temporary token after login |
| **Preview** | Data preview before full generation |
| **Self-healing** | Agent auto-recovery via ChromaDB RAG |
| **Task ID** | Celery task UUID for status and download |
| **Webhook** | HTTP POST notification on report completion |
